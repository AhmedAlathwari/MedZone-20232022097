<h2>📌 Project Overview</h2>

<p>
MedZone is an online pharmacy e-commerce platform developed using Laravel.
</p>

<p>
The system allows customers to browse medicines, view product details,
submit reviews, place orders, and manage their shopping experience.
</p>

<p>
Administrators can manage products, categories, users, reviews, messages,
FAQs, orders, and system settings through an admin dashboard.
</p>
